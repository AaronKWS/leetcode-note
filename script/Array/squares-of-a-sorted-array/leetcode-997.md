# 997. 有序数组的平方
### 给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。
leetcode 24题[>>>点这里查看](https://leetcode-cn.com/problems/remove-element/)
**示例：**
> **输入**: nums = [-4,-1,0,3,10]           
> **输出**: [0,1,9,16,100]                     
> **解释**: 平方后，数组变为 [16,1,0,9,100], 排序后，数组变为 [0,1,9,16,100]

**示例 2:**
> **输入**: nums = [-7,-3,2,3,11]       
> **输出**: [4,9,9,49,121]       

**提示：**
1. 1 <= nums.length <= 10^4
2. -10^4 <= nums[i] <= 10^4
3. nums 已按 非递减顺序 排序
