# 21. 合并两个有序链表
### 将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 
### 有效字符串需满足：
### 1.左括号必须用相同类型的右括号闭合。
### 2.左括号必须以正确的顺序闭合。
leetcode 21题[>>>点这里查看](https://leetcode-cn.com/problems/merge-two-sorted-lists/)

**示例：**
> **输入**: l1 = [1,2,4], l2 = [1,3,4]           
> **输出**: [1,1,2,3,4,4]            

**示例 2:**
> **输入**: l1 = [], l2 = []       
> **输出**: []       

**提示：**
1. 两个链表的节点数目范围是 [0, 50]
2. -100 <= Node.val <= 100
3. l1 和 l2 均按 非递减顺序 排列
