# 3. 无重复字符的最长子串
### 给定一个字符串**s**，请你找出其中不含有重复字符的**最长子串**的长度。
leetcode 3题[>>>点这里查看](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)

**示例：**
> **输入**: s = "abcabcbb"           
> **输出**: 3             

**示例 2:**
> **输入**: s = "bbbbb"       
> **输出**: 1       

**提示：**
1. 0 <= s.length <= 5 * 10^4
2. **s**由英文字母、数字、符号和空格组成
