# 5. 最长回文子串
### 给你一个字符串**s**，找到**s**中最长的回文子串。
leetcode 5题[>>>点这里查看](https://leetcode.cn/problems/longest-palindromic-substring/)

**示例：**
> **输入**: s = "babad"           
> **输出**: "bab"            

**示例 2:**
> **输入**: s = "cbbd"       
> **输出**: "bb"       

**提示：**
1. 1 <= s.length <= 1000
2. s 仅由数字和英文字母组成
