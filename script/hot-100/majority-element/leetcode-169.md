# 169. 多数元素
### 给定一个大小为 n 的数组 nums ，返回其中的多数元素。多数元素是指在数组中出现次数 大于 ⌊ n/2 ⌋ 的元素。
### 你可以假设数组是非空的，并且给定的数组总是存在多数元素。

leetcode 169题[>>>点这里查看](https://leetcode-cn.com/problems/linked-list-cycle/)
**示例：**
> **输入**: nums = [3,2,3]           
> **输出**: 3            

**示例 2:**
> **输入**: nums = [2,2,1,1,1,2,2]       
> **输出**: 2                    

**提示：**
1. n == nums.length
2. 1 <= n <= 5 * 10^4
3. -10^9 <= nums[i] <= 10^9
