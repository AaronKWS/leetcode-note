# 70. 爬楼梯
### 假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

### 每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？
leetcode 70题[>>>点这里查看](https://leetcode-cn.com/problems/climbing-stairs/)
**示例：**
> **输入**: n = 2           
> **输出**: 2            
> **解释**: 有两种方法可以爬到楼顶。
> 1. 1 阶 + 1 阶
> 2. 2 阶         

**示例 2:**
> **输入**: n = 3       
> **输出**: 3          

**提示：**
1. 1 <= n <= 45
